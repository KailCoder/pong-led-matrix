# Pong LED Matrix (Proteus 8.17)

Proyecto: Pong simplificado implementado con lógica discreta y matriz LED (8×8).  
Contiene BOM, pinout y guía paso a paso para montar y simular en Proteus 8.17.

Estructura del repositorio:
- README.md (este archivo)
- BOM.md (lista de componentes con nombres de Proteus 8.17)
- SCHEMATIC_PINOUT.md (conexiones pin a pin para chips usados)
- EQUATIONS.md (mapa de memoria, actualización de bola y paleta, FSM)
- INSTRUCTIONS.md (guía paso a paso para montar y simular en Proteus)

Objetivo del proyecto:
- Demo visual y fiable en ~3 días (6 horas aprox.) con hasta 2 h/día.
- MVP: matriz 8×8 multiplexada, bola (1 pixel), dos paletas (lado izquierdo y derecho), controles por botones, lógica de colisión básica y reinicio/puntuación simple.
